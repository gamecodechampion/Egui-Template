# Egui Template

The title is quite self-explanatory, this is just a simpler, more cutdown, easy to understand template for Egui to help you get started with your next app!


# Getting Started

## Prerequisites
To be able to edit this tool, it is suggested you have basic knowledge in the Rust programming language. Here are some official sources to help you learn rust if you need it:

* [Official Rust Handbook](https://doc.rust-lang.org/stable/book/)
* [Rustlings](https://rustlings.rust-lang.org/)

You must also have the [language](https://rust-lang.org/learn/get-started/) installed onto your system (Provided in the hyperlink).

## Use
To use this for yourself, simply just clone the project with `https://github.com/gamecodechampion/Egui-Template.git`, then edit it as you please!
